# Referencia Rapida: Devolucao no CRM

## Acoes Rapidas

### Realizar devolucao

1. Abra o item de origem (Pedido ou Demanda)
2. Clique em Devolver para...
3. Informe o motivo (obrigatorio)
4. Selecione o status de destino
5. Confirme a operacao

### Encontrar itens devolvidos

1. Abra o item de origem ou destino
2. Use os indicadores de relacionamento
3. Consulte o historico de devolucao no proprio item

### Ver itens finalizados

1. Ative Mostrar Finalizados
2. Revise os itens com status finalizador
3. Desative para voltar a visao operacional

## Tipos de Devolucao

| Origem  | Destino      | Permissao                            |
| ------- | ------------ | ------------------------------------ |
| Pedido  | Oportunidade | `crm:devolucao:order_to_opportunity` |
| Demanda | Pedido       | `crm:devolucao:post_sales_to_order`  |

## Controles de Visibilidade

| Controle            | Funcao                        |
| ------------------- | ----------------------------- |
| Mostrar Finalizados | Exibe/oculta itens concluidos |

## Indicadores Visuais

| Indicador               | Significado                          |
| ----------------------- | ------------------------------------ |
| Icone de devolucao      | Item com relacionamento de devolucao |
| Links de relacionamento | Navegacao entre origem e destino     |
| Cor diferenciada        | Item finalizado                      |

## Solucao Rapida de Problemas

### Nao vejo o botao de devolucao

- Verifique suas permissoes
- Refaca login
- Recarregue a pagina

### Erro ao confirmar devolucao

- Verifique o motivo preenchido
- Verifique o status de destino
- Tente novamente em alguns segundos

### Item nao aparece apos devolucao

- Verifique o status de destino escolhido
- Recarregue a esteira de destino
- Abra o historico de relacionamento do item de origem

## Boas Praticas

### Faca

- Descreva o motivo com clareza
- Escolha o status de destino corretamente
- Comunique a equipe impactada

### Nao faca

- Motivo generico
- Devolucao para corrigir detalhes simples
- Devolucao sem avaliar impacto no fluxo
